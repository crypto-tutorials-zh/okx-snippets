# OKX KYC Verification Reference Guide

## KYC Tier Levels & Limits

| Tier | Name | Requirements | Withdrawal Limit | Use Case |
|------|------|--------------|------------------|----------|
| Level 1 | Basic Verification | Email + Phone | Limited | View balances, place orders (fiat disabled) |
| Level 2 | Intermediate | Level 1 + ID Doc (passport/national ID) | Standard daily limit | Full trading, fiat deposits/withdrawals |
| Level 3 | Advanced | Level 2 + Residence Proof + Source of Funds | Highest daily limit | Large withdrawals, institutional features |

## Supported Identity Documents by Region

### Asia-Pacific
- Passport (all countries)
- National ID / Resident ID (China, Japan, Singapore, Hong Kong)
- Driver's License (supported in select countries)

### Europe
- Passport
- EU National ID Card
- Residence Permit

### Americas
- Passport
- Driver's License (US, Canada)
- National ID (Mexico, Brazil)

### Africa & Middle East
- Passport
- National ID
- Residency Permit

**Note:** Self-issued or expired documents rejected. Digital copies must be clear, legible, all four corners visible.

## Verification Timeline & Status Codes

| Stage | Typical Duration | Status in App |
|-------|------------------|---------------|
| Submission received | Immediate | "Under Review" |
| Initial scan (OCR + fraud check) | 5–30 minutes | "Processing" |
| Human review | 1–24 hours | "Pending Review" |
| Approval or rejection | Same day | "Approved" / "Rejected" |
| Re-submission after rejection | 24–48 hours | "Resubmitted" |

## Common Rejection Reasons & Fixes

| Reason | Solution |
|--------|----------|
| Document image blurry/cut off | Re-upload with all edges visible, good lighting, no glare |
| Name mismatch (legal vs. registered) | Provide official name change certificate or use name on ID |
| Address not visible on document | Ensure address line fully visible; if not, provide separate address proof (utility bill) |
| Selfie fails liveness check | Clear face, neutral expression, good lighting, no masks/glasses |
| Country/jurisdiction not supported | Contact support; some regions restricted by regulation |
| Multiple accounts flagged | Close duplicate accounts; one person = one verified account |

## Privacy & Security Best Practices

- OKX stores encrypted documents; access restricted to compliance officers
- Never share verification links via email/SMS—only use official OKX app.
- 2FA mandatory after Level 2 approval.
- Document retention: 7 years for regulatory compliance.
- Data breach contact: support@okx.com with "Security Incident" subject.

## Region-Specific Notes

- **China:** Requires mainland ID or Hong Kong ID; PRC residents may face withdrawal caps due to regulatory limits.
- **US:** Additional FinCEN check; Level 2+ required for staking features.
- **EU:** GDPR compliance; data deletion request honored within 30 days post-account closure.
- **Japan:** Japan residents can verify but subject to local exchange regulations.

## FAQ Quick Links

**Q: Can I use a temporary/foreign ID?**  
A: Temporary IDs accepted if government-issued and unexpired. Foreign passports always accepted.

**Q: How long does resubmission take after rejection?**  
A: 24–48 hours; address all noted issues in updated submission.

**Q: Is Level 3 required for futures trading?**  
A: Level 2 minimum; Level 3 recommended for withdrawal limits >$100k/day.

**Q: What if my country is not listed?**  
A: Contact OKX support; some regions restricted by compliance but may be added.
